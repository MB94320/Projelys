Contexte & architecture qualité

Context & quality architecture

La page Qualité ISO 9001 est le hub central du module qualité. Elle agrège en un seul écran les 3 dimensions de la qualité projet : livrables, non-conformités et audits — pour l'ensemble du portefeuille.

The ISO 9001 Quality page is the central hub of the quality module. It aggregates in a single screen the 3 dimensions of project quality: deliverables, non-conformities and audits — across the entire portfolio.

📐 OTD & OQD pondérés

📐 Weighted OTD & OQD

OTD = Σ(otd_i × totalDel_i) / Σ(totalDel_i)

Chaque projet contribue proportionnellement à son nombre de livrables. Un projet avec 50 livrables pèse 5× plus qu'un projet avec 10.

Each project contributes proportionally to its deliverable count. A project with 50 deliverables weighs 5× more than one with 10.

⏱ DoD global

⏱ Global DoD

DoD global = Σ(dodTotal_i) sur projets filtrés

Contrairement aux autres KPIs, le DoD global est une somme (pas une moyenne) de tous les retards cumulés du portefeuille, en jours.

Unlike other KPIs, the global DoD is a sum (not an average) of all cumulated delays across the portfolio, in days.

📐 Calcul NC ouvertes & fermées

📐 Open & closed NC calculation

ncOpen = Σ(p.ncOpen) projets filtrés
ncTotal = Σ(p.ncTotal) projets filtrés
ncClosed = ncTotal – ncOpen

Les NC "Annulées" sont incluses dans ncOpen (statut ≠ "Clôturé").

Cancelled NCs are included in ncOpen (status ≠ "Closed").

📐 Calcul délai NC & score audit

📐 NC delay & audit score calculation

Délai moy. = moy(avgCloseDelayNc) pour projets avec délai > 0 Avg delay = avg(avgCloseDelayNc) for projects with delay > 0

Score audit = moy(auditScore) pour projets avec score non null Audit score = avg(auditScore) for projects with non-null score

Les projets sans délai NC ou sans audit sont exclus de ces moyennes.

Projects without NC delay or without audit are excluded from these averages.

💡 Spécificité du DoD tableau

💡 Table DoD specificity

Dans le tableau projets, le badge DoD utilise le champ p.dod (retard moyen du projet), tandis que le KPI global utilise dodTotal (somme cumulée). Ce sont deux valeurs différentes :

In the project table, the DoD badge uses the p.dod field (project average delay), while the global KPI uses dodTotal (cumulative sum). These are two different values:

p.dod = retard moyen du projet (jours) p.dod = project average delay (days)
p.dodTotal = somme des retards du projet p.dodTotal = sum of project delays

⚠️ NC totales — lecture du badge

⚠️ Total NCs — badge reading

Le badge "NC projet" affiche le format N (crit: C) où N = total et C = critiques. La couleur est déterminée par le nombre de critiques (pas le total).

The "Project NCs" badge displays the format N (crit: C) where N = total and C = critical. The color is determined by the number of critical NCs (not the total).

Ex : "14 (crit: 0)" → Vert Ex: "14 (crit: 0)" → Green
Ex : "14 (crit: 3)" → Rouge Ex: "14 (crit: 3)" → Red

Bonnes pratiques — Hub qualité

Best practices — Quality hub

🗓 Revue mensuelle portefeuilleMonthly portfolio review

Ouvrir /quality en début de mois, appliquer le filtre "Statut: En cours", parcourir les alertes par colonne. Cibler les projets avec badge rouge sur OTD, NC ouvertes > 3 ou score audit < 70%.

Open /quality at the start of each month, apply "Status: In progress" filter, browse alerts by column. Target projects with red OTD badge, open NCs > 3 or audit score < 70%.

⚠️ Prioriser les NC critiquesPrioritize critical NCs

Le badge "NC projet" (crit: N) permet d'identifier instantanément les projets avec des NC critiques. Cliquer sur "NC" dans le tableau pour accéder aux détails et lancer un QRQC / 8D.

The "Project NCs" badge (crit: N) instantly identifies projects with critical NCs. Click "NC" in the table to access details and launch a QRQC / 8D.

📊 Utiliser le filtre chef de projetUse the PM filter

Le filtre "Chef de projet" permet à chaque manager de consulter uniquement son portefeuille. Idéal pour les revues individuelles hebdomadaires ou les comités de direction.

The "Project manager" filter lets each manager view only their portfolio. Ideal for individual weekly reviews or management committees.

📥 Export pour les comitésExport for committees

L'export CSV contient tous les projets (non filtré). Ouvrir dans Excel, ajouter un filtre automatique sur la colonne "status" pour préparer les comités de direction qualité.

The CSV export contains all projects (unfiltered). Open in Excel, add an auto-filter on the "status" column to prepare quality management committees.

🔗 Navigation croisée efficaceEffective cross-navigation

Les 3 boutons Actions (Livrables / NC / Audit) dans chaque ligne du tableau ouvrent directement le module concerné filtré sur ce projet. C'est le chemin le plus rapide vers l'analyse détaillée.

The 3 Action buttons (Deliverables / NC / Audit) in each table row directly open the relevant module filtered on that project. It's the fastest path to detailed analysis.
